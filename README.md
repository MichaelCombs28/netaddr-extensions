netaddr-extensions
==================

A few utilities that extend the functionality of netaddr library,
mostly grabbing irregular cidr ranges, and a few other things.

## Installation
```pip install git+https://github.com/MichaelCombs28/netaddr-extensions.git```


## Documentation
